This project focuses on building an interactive healthcare analytics dashboard using the Synthea EHR (Electronic Health Records) dataset in FHIR JSON format. The raw nested JSON files were processed and flattened using Python and Pandas to create analysis-ready datasets for Patients, Conditions, and Observations.

The cleaned datasets were visualized in Looker Studio to create a Population Health Dashboard that analyzes patient demographics, chronic disease prevalence, BMI trends, and glucose level trends over time. Interactive filters such as year, gender, age group, and condition name were added to improve exploration and usability.

Tools & Technologies Used:

* Python
* Pandas
* JSON / FHIR Data Processing
* Google Sheets
* Looker Studio

Key Insights:

* Older age groups showed a higher prevalence of chronic conditions.
* Hypertension were among the most common conditions observed.
* BMI and glucose trends showed noticeable variation over time across patient groups.
* Interactive analysis helped identify patterns across demographics and clinical observations.
